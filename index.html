Essential Vocabulary Word List

Created by boyeonihn

Usage:

If you would like to access the entire api, please use this URL reqest below:

https://essential-vocabulary-api.herokuapp.com/api/

If you would like to access a random word, please use this URL:

https://essential-vocabulary-api.herokuapp.com/random

Usage : The basic syntax of a URL request to the API is shown below:

https://essential-vocabulary-api.herokuapp.com/api/word/(word)

As an example, to get definition of the word "Abate", you can send request to:

https://essential-vocabulary-api.herokuapp.com/api/word/abate
